Plano de Treino Semanal de 5 Dias com Supersets

Um plano de treino focado em eficiência, utilizando supersets com exercícios não antagonistas para maximizar a intensidade e reduzir o tempo na academia. Clique no nome de um exercício para ver imagens de como executá-lo.

Avisos Importantes

  • Aquecimento: Sempre comece com 5-10 minutos de cardio leve e aquecimentos dinâmicos.
  • Técnica Primeiro: A prioridade é a execução correta. Use um peso que você possa controlar.
  • Descanso: Faça o mínimo de descanso entre os exercícios A e B do superset. Descanse 60-90 segundos após completar o par.
  • Nutrição e Sono: São fundamentais para a recuperação e resultados.

Dia 1: Peito e Bíceps

Superset Exercício (Clique para ver imagem) Séries x Reps
1 1A: Supino Reto com Barra ou Halteres 4 x 8-12
1B: Rosca Direta com Barra ou Halteres
2 2A: Supino Inclinado com Halteres 3 x 10-12
2B: Rosca Martelo
3 3A: Crucifixo na Máquina ou com Halteres 3 x 12-15
3B: Rosca Concentrada

Dia 2: Pernas (Quadríceps) e Abdômen

Superset Exercício (Clique para ver imagem) Séries x Reps
1 1A: Agachamento Livre com Barra 4 x 8-12
1B: Prancha (Plank) 4 x 30-60 seg
2 2A: Leg Press 45° 3 x 10-15
2B: Elevação de Pernas 3 x 15-20
3 3A: Cadeira Extensora 3 x 12-15
3B: Abdominal Crunch na Máquina 3 x 15-20

Dia 3: Costas e Tríceps

Superset Exercício (Clique para ver imagem) Séries x Reps
1 1A: Remada Curvada com Barra 4 x 8-12
1B: Tríceps Pulley com Corda
2 2A: Puxada Frontal (Pulldown) 3 x 10-12
2B: Tríceps Testa com Barra
3 3A: Remada Sentada na Máquina 3 x 12-15
3B: Mergulho no Banco 3 x até a falha

Dia 4: Ombros e Panturrilhas

Superset Exercício (Clique para ver imagem) Séries x Reps
1 1A: Desenvolvimento Militar com Halteres 4 x 8-12
1B: Panturrilha em Pé na Máquina
2 2A: Elevação Lateral com Halteres 3 x 12-15
2B: Panturrilha Sentado na Máquina
3 3A: Elevação Frontal com Halteres 3 x 12-15
3B: Crucifixo Invertido com Halteres

Dia 5: Pernas (Posteriores) e Corpo Inteiro

Superset Exercício (Clique para ver imagem) Séries x Reps
1 1A: Stiff / Levantamento Terra Romeno 4 x 10-12
1B: Flexão de Braço (Push-up) 4 x até a falha
2 2A: Cadeira Flexora 3 x 12-15
2B: Puxada no Remo (Máquina) 3 x 200m (tiro)
3 3A: Afundo com Halteres (Passada) 3 x 10-12 (por perna)
3B: Abdominal Remador 3 x 15-20

Lembre-se: consulte sempre um profissional de educação física. Bom treino!
